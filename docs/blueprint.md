# **App Name**: ChefPro

## Core Features:

- Gestão de Mesas e Pedidos: Permite registrar pedidos por mesa, adicionar e remover itens do menu, e acompanhar o status de cada pedido (pendente, em preparo, servido).
- Controle de Estoque de Ingredientes: Gerencia os níveis de estoque de todos os ingredientes, define pontos de reposição e emite alertas automáticos para itens em baixa.
- Cadastro e Gestão de Funcionários: Ferramenta para registrar e gerenciar informações de funcionários, incluindo atribuição de perfis de acesso e dados de contato.
- Sistema de Acesso Baseado em Perfis: Define permissões específicas para cada perfil (ex: garçom, gerente), garantindo acesso apenas às funcionalidades relevantes para cada função.
- Menu Digital e Atualização em Tempo Real: Oferece uma interface para criar e gerenciar o cardápio digital do restaurante, com atualizações instantâneas de itens, preços e descrições.
- Dashboard de Produção da Cozinha: Um painel visual para a equipe da cozinha acompanhar o fluxo de pedidos em tempo real, gerenciar o status de preparo e otimizar a expedição.
- Assistente de Sugestões de Vendas (IA): Uma ferramenta de IA que analisa dados de pedidos passados e atuais para sugerir pratos populares, combinações de itens e oportunidades de upsell personalizadas aos clientes.

## Style Guidelines:

- Cor primária: Um verde vibrante e limpo (#2EB85C) para elementos interativos e branding, evocando frescor e eficiência. Esta cor oferece um contraste eficaz em um esquema de cores claras.
- Cor de fundo: Um tom de branco com um sutil toque de verde (#F0F4F2) para a maior parte da interface, garantindo uma estética clean e espaçosa que remete à higiene e ordem.
- Cor de destaque: Um amarelo-esverdeado luminoso (#C9E052) para chamar a atenção para notificações importantes, estados de sucesso e outros elementos que necessitam de um contraste alto.
- Cores neutras: Utilização de uma paleta de tons de cinza (#333333 para texto principal e #666666 a #EEEEEE para secundários) para legibilidade de texto, divisores e fundos de cards, mantendo o visual profissional e discreto.
- A fonte 'Inter' (sans-serif) será utilizada para todos os textos. Sua legibilidade e estética moderna se alinham perfeitamente com a proposta de um sistema funcional e profissional, adequada tanto para títulos quanto para o corpo do texto.
- Ícones em estilo 'outline' ou minimalistas 'filled' que representem claramente as funcionalidades (ex: mesa, prato, chef, estoque), contribuindo para uma interface intuitiva e despoluída.
- Layouts baseados em grid, com bastante espaço em branco, priorizando a organização e a facilidade de escaneamento das informações. Design responsivo para garantir usabilidade em diferentes dispositivos, otimizando o fluxo de trabalho.
- Animações sutis e rápidas (ex: transições ao adicionar itens, alterar status de mesa) para feedback instantâneo ao usuário, melhorando a experiência sem causar distrações ou atrasos na operação.
